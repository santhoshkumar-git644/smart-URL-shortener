# Smart URL Shortener

A MERN stack URL shortener platform with React, Node.js, Express, MongoDB, and Redis.

## Features
- URL shortening
- Base62 Encoding
- Analytics tracking
- Custom aliases
- Authentication (JWT + bcrypt)
- Rate limiting
- Redis caching

## Tech Stack
- Frontend: React, Tailwind CSS (Vercel)
- Backend: Node.js, Express (Render)
- Database: MongoDB Atlas
- Cache: Redis

## Setup
1. `cd server` & `npm install`
2. `cd client` & `npm install`
